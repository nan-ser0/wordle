## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm start

# build for production
$ npm run build

# generate static project
$ npm run generate
```

For detailed explanation on how things work, check out the [documentation](https://reactjs.org/).
